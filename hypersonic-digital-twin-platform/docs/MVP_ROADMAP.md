# MVP Roadmap

## 24-Hour Hackathon Plan

Hour 0-3:
- Finalize architecture slide and folder structure.
- Wire FastAPI health, simulation start, TPS evaluate.

Hour 3-8:
- Implement simplified TPS formulas.
- Add WebSocket telemetry tick stream.
- Build dashboard layout and live charts.

Hour 8-14:
- Add 3D/thermal visual overlay.
- Add AI mock recommendations.
- Add plugin loader and one visible plugin.

Hour 14-20:
- Add report generation template.
- Add sustainability score and risk alerts.
- Polish UI labels and mission-control flow.

Hour 20-24:
- Record demo flow.
- Prepare architecture diagram.
- Test offline fallback and stable seed values.

## 48-Hour Upgrade Plan

- Add sample telemetry replay mode.
- Add CSV/JSON report export.
- Add material comparison panel.
- Add anomaly detection chart.
- Add Docker Compose.
- Add second plugin: adaptive materials.

## Future Expansion Roadmap

- Replace formulas with trained surrogate models.
- Persist telemetry in TimescaleDB.
- Add multi-mission comparison.
- Add cloud GPU inference.
- Add distributed optimization sweeps.
- Add user accounts and saved vehicle designs.
